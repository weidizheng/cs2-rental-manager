# 🔪 CS2 饰品出租管理终端 (CS2 Rental Manager) v2.1

> 开发、迁移与排障请阅读 [MAINTENANCE.md](MAINTENANCE.md)。其中记录的是当前实际实现、私密数据迁移和接口缓存策略；本文仅保留项目概览。

一个专为 CS2 饰品出租玩家设计的多平台自动化资产管理与账单追踪桌面客户端（基于 Python + PySide6）。

支持 **ECOSteam、C5GAME、IGXE、悠悠有品** 等平台的数据对接与大盘行情监控，具备 **SQLite 数据库与本地 JSON 文件的双向实时同步**、**多普勒 Phase 相位渲染**、**12 小时到期倒计时预警** 以及 **AI 账单截图 OCR 解析** 功能。

---

## ✨ 核心功能亮点

- **📊 资产总览看板**：实时计算买入总资产、当前每日净收益、在租/在库比例及整盘估算年化率。
- **🔄 数据库与 JSON 自动双向同步**：UI 界面的新增、修改、删除操作在更新 SQLite 数据库的同时，无感自动写回 `data/items.json`。
- **⏰ 12 小时倒计时预警**：租期到期剩余小时数 $\le 12$ 小时时，状态栏自动高亮为警示红字并带有 `⏰` 提醒图标。
- **🏷️ 多普勒相位 (Phase) 专属标注**：支持 P1/P2/P3/P4 及宝盒（Ruby/Sapphire/Emerald）等高级相位的视觉高亮展示。
- **🔍 跨平台市场大盘查询**：可同时查阅 ECOSteam 与 CSQAQ 的全网起租价、求购最高价及多平台底价。
- **🤖 AI 截图智能录入**：配套标准的 Prompt 指令，发截图给 AI 即可自动提取账单并直接生成导入 JSON。
