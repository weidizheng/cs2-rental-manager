# 浏览器订单同步扩展

此扩展只能与本项目正在运行的桌面程序配对。它不上传数据到任何远程服务；只在桌面程序明确请求后，从已打开的订单页读取可见文本并发送到 `127.0.0.1:8765`。

## 安装

1. 用 Chrome 或 Edge 打开 `chrome://extensions`（Edge 可用 `edge://extensions`）。
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”，选择项目中的 `browser-extension` 文件夹。
4. 启动桌面程序，在“资产与出租管理”点击“复制扩展配对令牌”。
5. 点击浏览器工具栏中的“CS2 出租订单同步”扩展图标，粘贴令牌并点击“保存令牌”。

令牌仅保存在当前浏览器的扩展本地存储与桌面程序的 `private-data/configs.json` 中。不要把它发给其他人。

## 使用 C5 同步

1. 使用日常浏览器登录 C5，并打开“出租记录”页面。
2. 在桌面程序点击“通过浏览器同步 C5”。
3. 扩展会在最多约 30 秒内读取当前已打开的 C5 订单页，并发送给本地程序；也可点击扩展的“立即检查同步请求”立即执行。
4. 桌面程序会按订单号去重保存订单，再按标准化后的磨损值（float）关联资产。相同 float 的最新订单显示在资产页，较早订单保留在历史中。

如果 C5 出现安全验证，扩展只会上报状态，不会绕过或自动点击验证。请在该浏览器标签页中手动完成验证，然后再次点击桌面程序中的同步按钮。

## 当前范围

- C5：页面文本已接入桌面程序解析和 float 订单跟踪。
- ECO、IGXE：扩展权限和默认浏览器订单页入口已准备好，但订单解析器尚未实现；需要根据实际订单页样本再接入。
- 扩展不会读取、导出或传输浏览器 Cookie、密码、完整浏览记录。
